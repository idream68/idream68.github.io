Flow语法结构

flow 语法其实是直截了当的，分为节点定义和节点连接两部分

节点定义

语法结构如下：X=>Y: Z
其中，X是变量名， Y是指操作模块名，冒号后面的Z是具体显示的文字内容。需要注意的是，冒号后要加空格才能识别，而X，Y与=>之间不允许有空格。
其中，变量名X和文字内容Z可以比较随意设置，但是Y是有固定的内容，主要有以下几种：

操作模块名 表示含义说明
start 开始
end 结束
operation 普通操作模块
subroutine 子任务块
condition 判断快
inputoutput 输入输出块
